## [Unreleased]

### Added
- Initial release of File Browser TUI
- Keyboard-driven navigation with vim-style (hjkl) and arrow keys
- Tabbed browsing with multiple directory support
- Split dual-pane view (Norton Commander style)
- Comprehensive undo/redo system with backup
- Content search (grep-like) with async progress reporting
- Bulk rename with pattern support (replace, numbered, regex, case, extension)
- Multi-selection with toggle, select all, invert
- Preview pane for text files
- Bookmarks with quick slots
- Windows drive support and UNC path handling
- Docker containerization
- Comprehensive test suite (412 tests across unit, integration, property, snapshot, and E2E)
